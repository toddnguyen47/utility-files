# Unity C Testing Framework
A simple project to try out Unity C Testing Framework.

# Prereqs
You'll need to install the following software:
- `gcc`
- `cmake`

# How to Run
- Simply run this script:
```bash
$ chmod +x cmakeBuildAndRun.sh
$ ./cmakeBuildAndRun.sh
```
